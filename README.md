## 🔑 Ключевые правила продукта

### Статусы пользователей
**Два пути роста:**
1. **Владелец с открытым рулём** → сразу **Premium** (∞ заявок, скидка 20%, приоритет в ленте)
2. **Пассажир / без руля** → Bronze → Silver → Gold → Premium (по количеству поездок)

| Уровень | Условие | Заявки за руль/день | Скидка |
|---------|---------|---------------------|--------|
| 🥉 Bronze | 0–5 поездок | 3/день | — |
| 🥈 Silver | 6–20 поездок | 5/день | 5% |
| 🥇 Gold | 21–50 поездок | 10/день | 10% |
| 💎 Premium | 51+ или владелец с рулём | без лимита | 20% |
| 👑 VIP | Назначение admin | без лимита | 30% |

**Лимиты:** только на бесплатные заявки «за руль» (дневной). Заявки пассажиром — без лимита. Платные заявки — без лимита.

**Статус = max(статус как водителя, статус как пассажира).**

### Классы опыта (новые)
🏁 Кольцо · 💨 Дрифт · 🌆 Город · 🏔 Бездорожье · 🚜 Спецтехника · 🏆 Спорт  
Мультиселект при создании объявления.

### VIP-скидка
VIP-пользователь получает **скидку 20%** на все поездки. Владелец получает компенсацию кредитами с платформы. Опция «Бесплатно для VIP» удалена.

### Заявки
- Подача заявки **бесплатна** (нет оплаты за заявку!)
- Платно = цена за саму **поездку**, которую устанавливает владелец
- Отзыв можно оставить через 1 день после поездки (уведомление в профиле)

### Бизнес-аккаунты
- Организации создают отдельные страницы (kartings, прокаты)
- Без владельца рядом — клиент управляет сам
- Залог — опционально
- Оплата: час / день / период
- Таблица рекордов кругов (для картинга/спорта)
- Не конкурируют с личными объявлениями в общем рейтинге

## 🗺 Ключевые сценарии для тестирования

1. **Регистрация с выбором страны** → onboarding → страна/город → личный/бизнес → разрешить руль = Premium
2. **Лента + VIP** → переключить роль → увидеть/скрыть VIP-метки и блок «Катаю бесплатно»
3. **Карточка машины** → лайк → заявка (бесплатно, без оплаты) → чат
4. **Бизнес** → фильтр 🏢 → Moscow Kart Club → рекорды → запись → поделиться
5. **Добавить транспорт** → 4 шага → классы → фото → бизнес/личное → ценообразование
6. **Профиль → Избранное** → машины добавленные через ❤️ в ленте
7. **Уровни** → два пути → дневные лимиты → смена демо-роли

- **Это прототип** — данные не сохраняются между сессиями

## 🚀 Система бустов и продвижения (v8)

### Пользователь
- `app/boost.html` — отдельная страница: выбор машины + плана, баланс кредитов, оплата
- `app/car-card.html` — кнопка **«Буст»** в нижней панели владельца; открывает встроенный модал с планами из Table API
- Параметр `?owner=0` в URL → режим гостя (нет кнопок редактирования и буста)
- По умолчанию (без параметра) → режим владельца (демо)

### Администратор (`admin/boosts.html`)
- Таблица всех планов из `tables/boost_plans` с сортировкой, фильтром и переключателями активности
- CRUD: создать / редактировать / удалить план; live-preview бейджа
- Поля плана: название, метка длительности, длительность в часах, стоимость в кредитах, эффект (топ / выделение / оба), бейдж + цвет, описание, порядок, активен
- Вкладка «Активные бусты» — список запущенных бустов с прогресс-барами, кнопка «Стоп»
- Гистограмма использования планов

### Table API: `boost_plans`
| Поле | Тип | Описание |
|------|-----|----------|
| `name` | text | Название плана |
| `duration_label` | text | Отображаемая метка («1 час», «7 дней») |
| `duration_hours` | number | Длительность в часах |
| `credits_cost` | number | Стоимость в кредитах |
| `effect` | text | `top` / `highlight` / `both` |
| `badge_label` | text | Текст бейджа на карточке |
| `badge_color` | text | Цвет бейджа hex |
| `description` | text | Описание для пользователя |
| `is_active` | bool | Показывать пользователям |
| `sort_order` | number | Порядок в списке |
