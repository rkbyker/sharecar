Ключевые сценарии для тестирования
User Flows
Пройди по этим сценариям чтобы проверить логику
  • 1 Новый пользователь: Онбординг → регистрация → добавить свою машину → попасть в ленту
  • 2 Найти машину мечты: Лента → нажать карточку → карточка Mercedes → подать заявку → видеть уровень одобрения
  • 3 Поиск по бренду: Поиск → выбрать BMW → выбрать модель → перейти на карточку
  • 4 Пассажир vs За руль: Карточка → кнопка «Подать заявку» → выбор типа: пассажир или водитель
  • 5 Владелец — управление: Кабинет владельца → входящие заявки → принять/отклонить → расписание
  • 6 Добавить машину: Добавить → 4 шага → тип доступа → инвайт / платный / бесплатный
  • 7 Чат — правило против торга: Чат → написать что-то про «дешевле» → увидеть предупреждение
  • 8 Модерация (Админ): Жалобы → посмотреть жалобу → заблокировать пользователя
  • 9 Выдать VIP (Админ): Пользователи → найти нужного → нажать «Назначить VIP»
  • 10 Заблокировать за нарушение (Админ): Жалобы и блок → посмотреть флаги → заблокировать
  • 11 Рейтинг участников: Рейтинг → переключить вкладки (баллы / за рулём / пассажиром) → тапнуть на участника → увидеть боттом-шит с детальной статистикой
  • 12 VIP-видимость в рейтинге: Рейтинг → демо-кнопка «Gold» → тапнуть на Александра К. → видишь только заглушку 🔒 → нажать «VIP» → тапнуть снова → видишь VIP-поездки
  • 13 VIP-секция на карточке машины: Карточка BMW M3 → блок «Кто катал» → по умолчанию 🔒 заглушка → кнопка «Сменить роль» → видишь катальщиков
  • 14 Уровни — смена роли: Уровни → кнопки 🥉🥈🥇💎👑 → видеть как меняется шпаргалка, прогресс и карточки
  • 15 Регистрация — выбор страны: Онбординг → Тип аккаунта (личный/бизнес) → Страна / Город → разрешить за руль = сразу Premium
  • 16 Бизнес — картинг: Лента → фильтр 🏢 Бизнес → Moscow Kart Club → Страница организации → таблица рекордов → запись → поделиться
Важное
Правило продукта: За руль может сесть только тот, кто добавил свою машину. Пассажиром может быть любой зарегистрированный пользователь. Никакой самостоятельной аренды — все поездки с владельцем.
🏢 Бизнес-аккаунты:
  • При регистрации — бесплатно. Позже переводятся на платную подписку.
  • Отображаются в общей ленте наравне с личными объявлениями.
  • Ранжируются и конкурируют только с другими бизнес-аккаунтами — не вытесняют личные объявления и не участвуют в их сортировке.
  • Могут создавать: картинг-страницы, прокат авто, спортивные организации.
  • Устанавливают ценообразование: почасово, посуточно или на период.
  • Опционально задают залог и режим «без присутствия владельца».
  • На странице организации: таблица лучших времён, отзывы участников, шеринг через ссылку и Telegram.
Серверы: Российский стек — Яндекс Облако / VK Cloud / Selectel. Раздел настроек и инфраструктуры открывается в Admin → Настройки.
Это прототип — данные не сохраняются. Все нажатия симулируют логику. Для рабочего alpha нужен бэкенд на российских серверах.
